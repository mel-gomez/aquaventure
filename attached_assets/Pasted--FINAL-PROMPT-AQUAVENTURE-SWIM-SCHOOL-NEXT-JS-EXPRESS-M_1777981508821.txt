✅ FINAL PROMPT – AQUAVENTURE SWIM SCHOOL (NEXT.JS + EXPRESS + MYSQL)

Create a modern, full-stack web application for Aquaventure Swim School, a children-focused swim school based in Angono, Rizal. The system should include a public website, swimmer/parent portal, and admin dashboard.

⚙️ Tech Stack (STRICT REQUIREMENTS)
Frontend: Next.js (latest, App Router)
Backend: Express.js on Node.js
Database: MySQL
ORM: Prisma
Authentication: JWT (access + refresh)
Styling: Tailwind CSS
Architecture: API-based (separate frontend/backend)
